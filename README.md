# hybrid-quantum-encoding

Hybrid quantum data encoding for mixed-type health datasets using PennyLane — comparative study of angle, amplitude, and basis encoding with VQC classification.

## Structure
- `data/` — UCI Obesity dataset
- `src/` — Preprocessing, circuit definitions, baseline
- `results/` — Experiment outputs and metrics
- `notebooks/` — Analysis notebooks

## Setup
```bash
pip install pennylane numpy pandas scikit-learn matplotlib
```
